# Legacy Prototype

This folder contains the original static prototype runtime.

It is the visual reference while the production React/Supabase app is built in `src/`.

Do not edit these files for new product architecture unless the visible prototype itself needs a design fix:

```text
styles.css
data.js
shared.js
pages.js
home.js
crash.js
spotlight.js
```

New maintainable code belongs in `src/`.
